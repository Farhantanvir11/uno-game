Add your sound files in this folder with these exact names:

- `button-press.mp3`
- `card-play.mp3`
- `draw-card.mp3`
- `invalid-move.mp3`
- `timer-tick.mp3`
- `win.mp3`
- `uno-call.mp3`
- `penalty.mp3`

Recommended:

- Format: `mp3`
- Length: `0.15s` to `1.5s`
- Keep files small for faster loading
- Use short UI/game SFX, not long music tracks

Where they are used:

- `button-press.mp3`: for Create Room, Join Room, and Start Game button presses
- `card-play.mp3`: when a card is successfully played onto the discard pile
- `draw-card.mp3`: when a card is drawn
- `invalid-move.mp3`: when a player tries to play an invalid card
- `timer-tick.mp3`: during the last 5 seconds of your turn
- `win.mp3`: when the game ends and the winner screen appears
- `uno-call.mp3`: when UNO is called
- `penalty.mp3`: when a penalty is applied
